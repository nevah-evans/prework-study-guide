HTML

  • The head element contains information about the website.
  • The body element represents the visible content shown to the user.
  • Metadata is information about accessibility, search engines, and performance, not typically seen by the users.
  • The title element shows up in the tab at the top of the browser.
  • Elements nested inside of other elements are called child elements. The elements that hold all of the children elements, such as the body tag, are known as the parent element.
  • "Elements" and "tags" are commonly used interchangeably, but there is a difference. an element referes to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • "H1"tags are for headers in on a webpage. "h1" is the biggest and "h6" is the smallest. There should only be one "h1" on each page.
  • The alt text is for screen reader and assistive technologies. It can also improve engine rankings for the website.
  • The "footer" element appears at the bottom of the webpage and usually contains the author, copyright, sitemap and navagation information.
  • The "p" element represents a paragraph or block of text.
  • The "ul", "ol", and "li" elements represent unordered lists (bullet points), ordered lists (numbered), and list items.
  • The "img" element holds information about an image and the "scr" attribute defines the location of the image.
  • The "br" element creates a line of empty space, or line break between blocks of content.
  • The "a" element stands for anchor. This creates links to the same webpage or other webpages. the "href" attribute points to the URL for the link.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS: Styles elements directly in the HTML file using specific tags and style attributes with CSS rule for a particular page element.
  • Internal CSS Style Sheet: Embeds an entire style sheet directly in the HTML file using a "style" element in the "head" element.
  • External CSS Style Sheet: The most common way to style a webpage. CSS rules are in a separate file, using a "link" element placed in the HTML's "head".
  • Declarations contain two components: the CSS property we want to apply and the value of the property.
  • Two selectors can follow the same CSS rule when using a comma. (EX. header,footer).
  • When stlying effects the X-axis is the value to the left and right of the element and the Y-axis is the value below the element.

Git

  • git status: check what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout main: switches back to the main code.
  • git pull main origin: uploads the (main) code to be worked on in VS Code.
  • git -A: commits all files in the working branch.
  • git commit -m "enter text": adds a message to the commit, used for describing changes and instructions on how to use application.
